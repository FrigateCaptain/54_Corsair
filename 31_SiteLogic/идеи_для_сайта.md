## Резюме постов в формате «Статья о том…» (Вариант B)

**Идея:** Генерировать краткое резюме текста поста с помощью LLM для полей `custom_excerpt` и `meta_description` в Ghost.

**Технические детали:**
- Формат резюме: «Статья о том, как…» или «Статья о том, что…», 1–2 предложения.
- Место внедрения: `scripts/odt_to_md.py` — в `generate_front_matter()` (сейчас excerpt берётся из первых 2–3 предложений текста, без LLM).
- Альтернатива: отдельный шаг после конвертации ODT → MD.

**Промпт-пример:**
```
Сделай краткое резюме текста в формате «Статья о том, как...». 1-2 предложения.

Текст:
{body_text[:4000]}
```

## Провайдеры LLM

| Провайдер | API | Ключ | Примечание |
|-----------|-----|------|------------|
| OpenAI | OpenAI API | Нужен | Облако, оплата по токенам |
| Anthropic | Anthropic API | Нужен | Облако, оплата по токенам |
| Ollama | http://localhost:11434/api/generate | Не нужен | Локально, офлайн после скачивания |

**Пример кода (Ollama):**
```python
def generate_summary_ollama(body_text: str) -> str:
    response = requests.post(
        "http://localhost:11434/api/generate",
        json={
            "model": "llama3.2",
            "prompt": f"Сделай краткое резюме текста в формате «Статья о том, как...». 1-2 предложения.\n\nТекст:\n{body_text[:4000]}",
            "stream": False,
        },
        timeout=60,
    )
    return response.json()["response"].strip()
```

## Дополнительные теги через LLM

**Идея:** LLM предлагает семантические теги по смыслу текста — как дополнение к хэштегам из `extract_hashtags()` в `scripts/odt_to_md.py`.

**Пример:** Пост «Изменение культуры» — хэштег `вестисполей` (из #вестисполей) + LLM предлагает `управление`, `культура` по содержанию. Итого: `tags: [вестисполей, управление, культура]`.

**Реализация:**
- Внедрение в `generate_front_matter()`: объединять результат `extract_hashtags(md_content)` с ответом LLM (запрос вида «Предложи 2–4 тега по смыслу текста»).
- `scripts/ghost_publish.py` уже передаёт теги в Ghost API корректно (строки 318–320).
- Видимость тегов на сайте зависит от темы Ghost (`post.tags` в шаблоне).
